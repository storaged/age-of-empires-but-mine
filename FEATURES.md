# MVP Features

## Core Gameplay

- Grid-based map
- Unit selection (click + drag)
- Unit movement
- Resource gathering
- Building placement
- Basic AI opponent
- Win condition (destroy base)

## UX Requirements

- Selection highlighting
- Movement indicators
- Smooth camera movement
- Responsive input
- Stable 60 FPS

## Out of Scope (for now)

- Multiplayer implementation
- Advanced pathfinding
- Complex UI
- Animations polish
